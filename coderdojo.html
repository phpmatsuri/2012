子供向けプログラミング教室 Coder Dojo@福岡 開催！

プログラミングに挑戦してみませんか? CoderDojoは小中学生を対象にした無料の「プログラミング道場」。
これまでも東京、沖縄などの各地で開催されてきました。東京での様子は、こちらの写真をどうぞ。
https://www.facebook.com/media/set/?set=a.367523619965341
https://www.facebook.com/media/set/?set=a.378002488917454

■参加資格
年齢 : 小中学生
定員 : 10人まで
※ 当日、保護者の付き添いが必要です。
※ お子さんの学年と、プログラミングの希望(あれば)をコメント欄に書いてください。


■タイムテーブル
11/4(日)
8:30 やる内容で班分け (ちょっと早めに集ってね!)
8:35 道場スタート
9:30 道場おわり。解散! 10:00から始まるPHPMatsuriコンテストも見学できます。
※道場のあとも作業OK。先生が残っていれば質問も。


■先生はどんな人?
プログラマやデザイナ、エンジニアなど、ソフトやハードを作ることを専門にしている人たちです。
こどものコンピュータ教育に携わる方、会場のOSSCafeの常連ボランティアさん他。
※1,2人のこどもに対して、先生が1人付きます。


■持ち物
ノートパソコン (Mac, Windows, Linux, など)
ノートパソコンの電源アダプタ- マウス (小学校低学年ほか、タッチパッドに不慣れな場合)


■内容
Scratch (低学年向け)
HTML + CSS + JavaScript (高学年向け)


■会場
PHPMatusriと同じ会場です
http://www.phpmatsuri.net/2012/access.html


■参加応募
ATNDよりご応募ください。先着順ですのでご注意ください。
